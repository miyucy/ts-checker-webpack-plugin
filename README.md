# ts-checker-webpack-plugin

## Usage

```js
const TsCheckerPlugin = require("ts-checker-webpack-plugin").default;

module.exports = {
  // :
  plugins: [
    new TsCheckerPlugin()
  ]
};
```

## Examples

```bash
git clone [this repo]
cd [repo]
yarn install
yarn build

cd examples/prj1
yarn install
yarn build
```
